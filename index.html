Steve Jobs

Fue un empresario, inventor y cofundador de Apple Inc, una de las compañías de tecnología más influyentes del mundo. Es conocido por su visión innovadora y su habilidad para crear productos revolucionarios como el iPhone, el iPod y la Macintosh. También fundó Pixar Animation Studios. Jobs dejó un impacto duradero en la industria tecnológica y es reconocido por su enfoque en el diseño y la experiencia del usuario.

Bill Gates

Es un empresario, filántropo e informático estadounidense. Es el cofundador de Microsoft, una de las mayores empresas de software del mundo. Ha contribuido significativamente al desarrollo de sistemas operativos y software de productividad como Windows y Microsoft Office. A través de su fundación, ha dedicado una gran parte de su riqueza a la filantropía, especialmente en áreas como la salud global y la educación.

Ada Lovelace

Fue una matemática y escritora británica del siglo XIX, reconocida como la primera programadora. Es conocida por sus notas sobre la Máquina Charles Babbage, en las que elaboró algoritmos destinados a ser procesados por la máquina. Su trabajo sentó las bases para el desarrollo de la programación de computadoras y su legado es conmemorado en el Día de Ada Lovelace, para reconocer la contribución de las mujeres en la tecnología y la informática.

Mar del Plata, ubicada en la provincia de Buenos Aires, Argentina, es una popular ciudad costera famosa por sus extensas playas y su vibrante escena turística. Ofrece una amplia gama de actividades recreativas, una rica oferta gastronómica y una animada vida nocturna. La ciudad también es conocida por su escena cultural, con teatros, festivales y eventos artísticos. Es un destino ideal para aquellos que buscan disfrutar del sol y el mar, así como de una experiencia urbana dinámica. Además, cuenta con una gran variedad de opciones de alojamiento, desde hoteles boutique hasta acogedores hostales, para satisfacer las preferencias de todo tipo de viajeros.

Desarrollo Web Full Stack

HTML

HTML es el lenguaje de marcado estándar utilizado para crear páginas web. Define la estructura y el contenido de una página web mediante etiquetas y elementos que indican cómo se deben mostrar los elementos en un navegador web.

CSS

CSS es un lenguaje de estilo que se utiliza para controlar el diseño y la presentación de una página web. Permite definir el aspecto visual de los elementos HTML, como el color, la fuente, el diseño y otros estilos.

JavaScript

JavaScript es un lenguaje de programación de alto nivel que se utiliza para agregar interactividad y dinamismo a las páginas web. Es ejecutado en el navegador del usuario y permite manipular el contenido de una página, responder a eventos y comunicarse con servidores.

Java

Java es un lenguaje de programación de propósito general que se caracteriza por su portabilidad y capacidad de ejecutarse en múltiples plataformas. Es ampliamente utilizado en el desarrollo de aplicaciones empresariales, aplicaciones móviles y software de servidor.

Spring Boot

Spring Boot es un marco de desarrollo de aplicaciones basado en Java que facilita la creación de aplicaciones empresariales robustas y escalables. Proporciona un conjunto de herramientas y convenciones que aceleran el proceso de desarrollo y simplifican la configuración.

Git

Git es un sistema de control de versiones que se utiliza para rastrear cambios en el código fuente durante el desarrollo de software. Permite a los equipos de desarrollo colaborar, llevar un historial de revisiones y gestionar el flujo de trabajo de desarrollo de manera eficiente.
